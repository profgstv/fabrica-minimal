# fabrica-minimal

**Modificação do tema es-theme-minimal, originalmente feito por [lilbud](https://github.com/lilbud/es-theme-minimal), para utilização no sistema Batocera.Linux adaptado para utilização nas Fáricas de Cultura 4.0 do Estado de São Paulo**

**Adaptação feita pelo Educador do Ateliê de Games da Fábrica de Cultura 4.0 de São Bernardo do Campo/SP, [Gustavo Jorge](https://github.com/profgstv)**

* Originalmente, o tema foi desenvolvido como uma tentativa de deixar o EmulationStation mais parecido com um sistema de jogos. Para fins educativos, promovemos uma estilização baseada também no tema Simple_Life, por [DarrenCarol](https://github.com/DarrenCarol/Simple_Life), deixando apenas um plano de fundo padrão com a identidade visual da Fábrica de Cultura.

* Além disso os ícones dos sistemas foram editados para uma tradução mais compreensível no português e para a melhor interação dos usuários.
